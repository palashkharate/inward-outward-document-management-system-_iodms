# ABSTRACT (Draft)

This internship report presents the technical work carried out during a one-month internship at Hindustan Aeronautics Limited (HAL), focusing on two complementary engineering domains: artificial intelligence for aircraft engine predictive maintenance and the design of an Inward/Outward Document Management System (IODMS).

The predictive maintenance study investigates Remaining Useful Life (RUL) estimation using the NASA C-MAPSS dataset. The workflow includes data preprocessing, exploratory data analysis, feature engineering, machine learning model development, performance evaluation, and deployment concepts. Results and discussion will use actual project outputs only.

The software engineering component documents the development of IODMS for secure document management in an air-gapped organizational environment. The report covers requirements engineering, system architecture, database design, V-Model development, frontend/backend implementation, deployment using Docker, and software verification.

Sections requiring organization-specific information remain clearly marked as placeholders and will be completed using verified internship records.
